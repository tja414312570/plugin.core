# plugin.core Plugin框架的核心
	* 提供配置上下文（ConfigContext）
	* 提供组件代理 （Proxy）
	* 提供调用栈信息记录 （InvokeStack）
	* 提供参数、方法、构造器等工具 （ParameterUtils）
	* 简单的注解方式 （@Register、@Service）
	* 提供错误处理
	* 提供依赖注入（DI @Service）
	* 基于方法、构造器、以及属性的拦截器（AOP）
	* 提供属性注入 （@Property）
	* 提供资源注入 （@Resource）
	* 提供bean管理与注入（和组件不同）
	* jdk和cglib代理以及双代理模式
	* 单例与非单例支持
	* 热更新功能（ClassHotUpdater）
	* 调用加密功能（@Encrypt）
